# この `docs` フォルダについて

このフォルダはリポジトリの構成・開発フロー・よく行う修正方法を分かりやすくまとめた説明用ドキュメントです。

ファイル一覧:

- `overview.md` — TypeScript / React / Next.js / Tailwind の基本知識
- `structure.md` — このリポジトリのディレクトリ構成と主要ファイル一覧
- `edit-examples.md` — よくある変更例と手順（パス付き）
- `how-to.md` — 開発 / ビルド / 記事追加などの実務コマンド
- `faq.md` — トラブルシューティングと注意点

まずは `structure.md` と `edit-examples.md` を読むことをおすすめします。
